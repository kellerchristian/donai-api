# DonAI API

Backend REST API for a blood donation platform.

This project is part of a full-stack application aimed at managing donors, blood donations, and requests in an organized and scalable way.

## 🚀 Tech Stack

* Kotlin
* Ktor
* Gradle (Kotlin DSL)
* Koin (Dependency Injection)

## 🧩 Features (planned)

* Donor management
* Donation tracking
* Blood request system
* RESTful API design
* Clean architecture structure

## 🏗️ Project Structure

The project follows a layered architecture:

* `routes` → HTTP endpoints
* `controllers` → request handling
* `services` → business logic
* `repositories` → data access
* `models` → data representation

## 📡 API Design

The API is designed to be independent of any frontend client and follows REST principles.

Example endpoints:

* `GET /donors`
* `POST /donations`
* `GET /requests`
